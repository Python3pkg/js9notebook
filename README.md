js9notebook
===========

Embed JS9 (the javascript version of DS9 (the astronomical image viewer))
inside Jupyter.


install
=======

From source:
```
$ python setup.py install
```

Or with pip:
```
$ pip install js9notebook
```

usage
=====
```
import js9notebook
js9notebook.initialize()
```
